
urdu.h is a C header library that allows programmers to write C code using Urdu keywords and syntax. It replaces traditional English-based constructs and functions with Urdu equivalents, making programming more intuitive and accessible for native Urdu speakers â€” especially beginners.


Installation

Clone the repository:
git clone https://github.com/yourusername/urdu.h.git

Include the header file:
#include "urdu.h"

Compile and run:
gcc demo.c -o demo
./demo
